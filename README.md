# Capturing Cell-Cell Interactions with cell2cell

## Installation
Create a new conda environment:
```
conda create -n cell2cell python=3.7
```

Go into cell2cell directory:
```
cd cell2cell
```

Then, install cell2cell:
```
pip install .
```
